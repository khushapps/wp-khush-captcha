== Description ==

This brand new type of Captcha protects your website from bots while making you and your website's users happy. Use this plugin tp prevent spam bots from registering a iser on your site. 

== Installation ==
1. Upload 'khush-captcha' folder to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Captcha will now show up on the registration page. 
1. You can change the 'Blank Captcha' and 'Capthca Error' messages under Settings > Khush-Capthca
