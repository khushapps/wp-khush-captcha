== Description ==

This plugin adds a new form of captcha to your registration and comment forms. 
Use this plugin to provide an "are you human" check on your site. 

== Installation ==
1. Upload 'khush-captcha' folder to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Captcha will now show up on the registration and comment forms. 
1. You can change the 'Blank Captcha' and 'Capthca Error' messages under Settings > Khush-Capthca
